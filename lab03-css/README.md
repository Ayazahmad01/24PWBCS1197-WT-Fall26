## Lab Objective
The objective of this lab is to understand and implement various **CSS styling techniques** including selectors, typography, backgrounds, layouts, animations, and responsive design to create visually attractive web pages.

## Tasks Completed
### Task 1 вЂ“ CSS Selectors Practice
Implemented different CSS selectors including element, class, ID, attribute selectors, combinators, pseudo-classes, and pseudo-elements to apply styles to HTML elements.

### Task 2 вЂ“ Typography Showcase
Demonstrated typography styling using different font families, font sizes, font weights, text alignment, spacing, text decoration, and text shadow effects.

### Task 3 вЂ“ Background Gallery
Created multiple background styles including solid colors, background images, gradients, and fixed backgrounds to demonstrate different CSS background properties.

### Task 4 вЂ“ Image Styling Portfolio
Designed an image gallery using CSS with rounded corners, circular images, shadows, hover effects, and responsive layout.

### Task 5 вЂ“ Button Styling Collection
Developed a collection of styled buttons with different sizes, colors, gradients, hover effects, transitions, and disabled states.

### Task 6 вЂ“ Form Styling
Styled a registration form using custom input fields, focus states, styled buttons, dropdown menus, and basic form layout improvements.

### Task 7 вЂ“ Card Component Design
Created product/service cards with images, descriptions, consistent spacing, shadow effects, hover transformations, and call-to-action buttons.

### Task 8 вЂ“ Navigation Menu Styling
Designed a navigation menu with horizontal layout, hover effects, active states, and responsive behavior.

### Task 9 вЂ“ CSS Transform Gallery
Demonstrated CSS transform effects including rotation, scaling, translation, skewing, transform origin, and combined transformations.

### Task 10 вЂ“ Advanced Effects Showcase
Implemented advanced CSS effects such as gradient text, custom checkboxes and radio buttons, loading animations, text clipping, and complex shadow effects.

### Task 11 вЂ“ Personal Portfolio Styling
Applied CSS styling to the Lab 02 portfolio including color scheme, typography improvements, spacing, navigation styling, and responsive design.

### Task 12 вЂ“ Course Website Styling
Styled the Lab 02 course schedule table using zebra striping, hover effects, custom borders, spacing, color-coded course types, and responsive table layout.

### Task 13 вЂ“ GitHub Pages Enhancement
Enhanced the GitHub Pages website with improved CSS styling, consistent colors, better typography, enhanced images, and a more professional appearance.
