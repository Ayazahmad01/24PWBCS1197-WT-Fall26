# Web Technologies Lab 03 & 04
## Student Information
Course: Web Technologies (CS224L)  
Lab: CSS Fundamentals  
## Tasks Completed
### Task 1 – CSS Selectors Practice
Different CSS selectors such as element, class, ID, attribute selectors, combinators, pseudo-classes, and pseudo-elements were implemented to style HTML elements.
### Task 2 – Typography Showcase
Various typography properties like font family, font size, font weight, alignment, spacing, text decoration, and text shadow were demonstrated.
### Task 3 – Background Gallery
Different CSS background properties including background color, background images, gradients, and fixed backgrounds were used to design webpage sections.
### Task 4 – Image Styling Portfolio
Images were styled using CSS effects such as rounded corners, circular shapes, shadows, and hover effects to create an attractive gallery layout.
